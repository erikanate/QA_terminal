Linux terminal (GitBash) commands

1. pwd - посмотреть где я
2. mkdir f_0 - создать папку
3. cd f_0 - зайти в папку
4. mkdir f_1 f_2 f_3 - создать 3 папки
5. cd f_1 - зайти в любую папку
6. touch test.txt test1.txt test2.txt test3.json test4.json - создать 5 файлов (3 txt, 2 json)
7. mkdir d_1 d_2 d_3 - создать 3 папки
8. ls - вывести список содержимого папки (не в столбик)
   ls -la - вывести содержимое папки в столбик со скрытыми файлами
   ls -l - вывести содержимое папки в столбик без скрытых файлов
9. cat test.txt - открыть любой txt файл
10. vim test.txt - написать туда любой текст
11. escape :wq enter - сохранить и выйти
12. cd .. - выйти из папки на уровень выше
13. mv test.txt test1.txt d_1 - переместить любые 2 файла в любую другую папку
14. cp test3.json test4.json d_2 - скопировать любые 2 файла в любую другую папку
15. find . -name "*.json" - найти файл по имени
16. tail -f test.txt - просмотреть содержимое в реальном времени
    grep Kate test.txt - найти строку со словом в тексте
    grep -n Kate test.txt - найти строку с номером строки и словом в тексте
17. head test.txt - вывести несколько первых строк файла
18. tail test.txt - вывести несколько последних строк файла
19. less -3 (любое необходимое число) test.txt - показывает 3 первые строки длинного файла
    less +3 (любое необходимое число) test.txt - просмотр длинного файлаБ начиная с 3й строки
20. date - вывести дату и время
